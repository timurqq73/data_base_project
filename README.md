# Проект: База данных интернет-магазина спортивного питания "ProteinPower"

## Описание проекта
Разработка реляционной базы данных для управления интернет-магазином спортивного питания. База данных будет охватывать все основные бизнес-процессы: учет товаров, управление поставщиками, обработку заказов, доставку и взаимодействие с клиентами.

## Технический стек

- SQL
